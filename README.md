# boot-basic

Spring Boot 학습용 프로젝트입니다.  
기본 CRUD + JPA + Thymeleaf + Spring Security(Form Login)까지 실습하며 정리합니다.

## Requirements
- Java 21
- Gradle
- Spring Boot (프로젝트 설정 기준)

## Run
```bash
./gradlew bootRun
```
- 접속: http://localhost:8080
